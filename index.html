Justin's HTML Cheat Sheet

Header

Tag Child of: Contents Description
<Head> <html> Title, Links, Meta Tells the browser how to interact with the file, links file to other files, and gives the page a browser title.
<Title> <Head> "Name of Webpage" Gives the page a title on the browser tab.
<Meta> <Head> charset="UTF-8", name="viewport" content="width=device-width, initial-scale=1.0", http-equiv="X-UA-Compatible" content="ie=edge" Defines the metadata of an html document.
<Link> <Head> rel="stylesheet" href="style.css" Links the html and css documents together.

Body

Tag Name Child of: Contents Purpose
<Body> Body <html> All content that appears on the actual web page, all child tags not included in the <head> section. Every child tag of this tag is what appears in the web browser.
<h1> Heading 1 <body> "Title" Usually signifies the main title of the webpage, typically only used once per website, and is the largest heading setting available.
<div> Divide <body> Any content to be seperated from other parts of the <body> tag. Often this tag is used to house CSS classes for seperate parts of a website.
<p> Paragraph <body>, <div>, <span>, <td>, <li> Any text that will appear on the web page as writing. This can be used as a full paragraph or a simple sentence to appear on the page. It can also be used in tables or lists.
<ul> Unordered List <body>, <span>, <div> Filled with stacked <li> tags that appear as bullet points one over another. Good for listing items (such as for a grocery list or recipe) that require no specific ordering.
<ol> Ordered List <body>, <span>, <div> Filled with stacked <li> tags that appear as a numbered list. Good for listing things that need to be done in a specific order or for rankings.
